# no-slop.ai

`no-slop.ai` is a small site about a simple rule: don't send AI-generated content you haven't read, edited, or understood.

Live site: [no-slop.ai](https://no-slop.ai/)

## What This Is

The site is a short public service announcement for anyone tired of being asked to read someone else's unedited model output.

Own your output.

## Inspiration

[nohello.net](https://nohello.net)

## Local Development

This repo uses Bun for package management and scripts.

```bash
bun install
bun run dev
```

Then open the local Vite URL printed in the terminal.

Useful scripts:

- `bun run dev` starts the local development server.
- `bun run build` creates a production build.
- `bun run preview` serves the production build locally.
- `bun run lint` runs ESLint.
- `bun run format` runs Prettier across the repo.
- `bun run deploy` builds the site and deploys it with Wrangler.

## Stack

- TanStack Start + React 19
- Tailwind CSS 4
- Vite
- Cloudflare deployment via Wrangler

Core page copy and layout currently live in `src/routes/index.tsx`, with shared site styling in `src/styles.css`.
